CSS flexbox — це CSS модуль. Flex дозволяє автоматично організовувати відповідні елементи в контейнері залежно від розміру екрана та дає змогу використовувати більш ефективний спосіб верстки, вирівнювання й розподілу вільного місця між елементами у контейнері, навіть коли їх розмір невідомий і/або динамічний.
